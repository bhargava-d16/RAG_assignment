# Swiggy Annual Report RAG QA System

This project implements a Retrieval-Augmented Generation (RAG) system to answer questions based on the Swiggy Annual Report.

## Features

- PDF document processing
- Text chunking and preprocessing
- Semantic embeddings
- FAISS vector database
- Retrieval-Augmented Generation (RAG)
- Question answering interface

## Tech Stack

- LangChain
- Groq LLM (Llama 3.3)
- FAISS
- HuggingFace Embeddings
- PyMuPDF

## Dataset Source

Swiggy Annual Report (Public Source)

https://www.swiggy.com/corporate/

## How to Run

1. Open the notebook in Google Colab
2. Upload the Swiggy Annual Report PDF
3. Enter Groq API key when prompted
4. Ask questions about the report
